Slipstream UI Widgets

Slipstream provides a comprehensive UI widget toolkit in order to provide consistency in look-and-feel and interactions between workflows. Workflow developers are strongly encouraged to use the widget toolkit in their implementations.

The toolkit contains the following widgets:

BarChart

Carousel

ConfirmationDialog

ContextMenu

Dashboard

Date Picker

Drop Down

Form

Grid

Ip Cidr

Layout

List Builder

New List Builder

Login

Map

Overlay

Progress Bar

Search

Short Wizard

Spinner

Tab

Time

Time Range

Time Zone

Time Series Chart

Tooltip

Tree

Slipstream UI widgets are typically utilized in Slipstream workflows, but the widgets can be used independently of the framework as well.
